абсолютно все пункты опциональны и корректируются. при желании студента можно перескочить на другую тему или вовсе перепрыгнуть на следующий раздел.

введение в CMS, Wordpress.

  1. 1. что такое CMS. локальные сервера. PHP. структура WP темы. хуки.
  2. 2. установка WP. натяжка темы WP. подключение скриптов и стилей. плагины. настройка заголовка, логотипов сайта.
  3. 3. работа с произвольными полями. динамический вывод постов.
  4. 4. посты и картинки-превью. карты в WP. натяжка форм. добавление reCAPTCHA. перенос натяжки на хостинг.

натяжка многостраничных сайтов.

  1. 1. шаблоны страниц. привязка к динамическому меню.
  2. 2. работа со страницами-постами. шаблоны страниц. работа с SEO. оптимизация и ускорение натяжки.

введение в JavaScript.

  1. 1. что такое js. для чего придумали и для чего нужен. синтаксис, все базовые конструкции. именование переменных.
  2. 2. массивы и методы массивов. объекты. работа с ними.
  3. 3. синтаксис ES6. замыкания, лексическое окружение и полная работа с функциями.
  4. 4. работа с DOM. получение элементов, изменение элементов.
  5. 5. навигация по DOM-элементам. data-атрибуты и остальная работа по DOM.

JavaScript в реальных проектах (pet-проекты).

  1. 1. реализация проекта с генерированием рандомного цвета.
  2. 2. реализация бургер-меню и модального окна.
  3. 3. работа с объектом Date. таймер.
  4. 4. использование классов (ООП) в реальной работе. практический проект с выводом карточек товаров через ООП.
